Detailed Description

template<class T, std::endian E>
requires (std::integral<T> || std::floating_point<T>)
struct nanom::endian_scalar< T, E >

be<T> / le<T> — an explicitly big/little-endian wire scalar.

Stored as raw bytes (alignment 1, packing-safe); converts to/from host order on access. Mixing be<> and le<> fields in one struct is fine.
