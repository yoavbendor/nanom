Detailed Description

template<Described Row>
class nanom::bulk_table< Row >

Column-major, index-addressed table: one contiguous buffer per flattened leaf column, pre-sized to N rows.

Reuses soa<Row>'s column layout (names, Arrow format strings, element widths). A validity byte per row records whether the kernel produced output (a packet that didn't match decodes to a default row with valid=0). The column buffers are exactly what nanoarrow / Lance import.
